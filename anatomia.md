# Anatomia

Toda propriedade é seguida de : (dois pontos) e um valor e um ; (ponto e vírgula) para encerrar essa ideia de valor.
Ex de css:

h1 {
    color: blue;
    font-size: 60px;
    background: black;
}

Os elementos do CSS são então:

Selectors (seletores): Nesse caso o h1, que vai buscar no HTML a tag h1 e aplicar as mudanças.

Declaration (declaração): As chaves {} e tudo dentro delas.

Properties (propriedades): As coisas a serem alteradas. O que vem antes dos : (dois pontos)

Property values (valores das propriedades): Os novos valores que estamos atribuindo a tais propriedades. O que vem depois dos : (dois pontos)
